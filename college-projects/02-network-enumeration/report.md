# Network Enumeration & OWASP

## Target
4-subnet corporate network environment

## Tools Used
Nmap, Wireshark

## What I Did
- Performed detailed network enumeration across 4 subnets
- Identified all active hosts, services, and vulnerabilities
- Applied OWASP Top 10 framework to identify web
  application vulnerabilities
- Documented findings with risk ratings

## Key Findings
Multiple web application vulnerabilities identified
across subnets. Several services running outdated
versions with known CVEs.

## What I Learned
OWASP Top 10 provides a practical framework for
identifying the most critical web application risks.
Enumeration depth directly impacts assessment quality.
