# Final Project — Full Security Engagement

## Target
Simulated enterprise environment — 4 targets

## Tools Used
Nessus, HoneyBOT, Kali Linux, MegaPing

## What I Did
- Performed complete vulnerability assessment across
  all enterprise targets
- Deployed and monitored HoneyBOT honeypot
- Analysed all findings and assigned CVSS scores
- Produced client-facing Executive Summary
- Produced full Vulnerability Remediation Report

## Results
| Severity | Count |
|----------|-------|
| Critical | Multiple |
| High | Multiple |
| Medium | Multiple |

## Deliverables
- Executive Summary for management audience
- Detailed Remediation Report for technical team
- Full vulnerability findings with CVSS scores
- Prioritised remediation recommendations

## What I Learned
A complete security engagement requires both
technical skill and professional communication.
The ability to explain technical findings to
a non-technical audience is as important as
finding the vulnerabilities themselves.
