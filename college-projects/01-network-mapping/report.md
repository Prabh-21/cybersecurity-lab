# Network Mapping & Compliance

## Target
Multi-subnet corporate network environment

## Tools Used
Nmap, Wireshark

## What I Did
- Performed network discovery across multiple subnets
- Identified active hosts, open ports, and running services
- Analysed findings against PIPEDA compliance requirements
- Documented security gaps and recommended remediation

## Key Findings
Unnecessary open ports and services exposing the network
to potential threats. Mapped full network topology
including all connected devices.

## What I Learned
Network mapping is the foundation of every security
assessment. Understanding what is on your network is
the first step to securing it.
